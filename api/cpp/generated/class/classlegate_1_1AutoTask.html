void add_constraints(Span<const Constraint> constraints)#

Adds multiple partitioning constraints to the task.

This function is a convenience when adding multiple constraints simultaneously.

task.add_constraints({c1, c2, c3});

Is functionally equivalent to:

task.add_constraint(c1);
task.add_constraint(c2);
task.add_constraint(c3);

This routine is most commonly used with the overloads of partitioning routines that return multiple constraints:

const auto cstrnt = legate::align({part1, part2, part3});

task.add_constraints(cstrnt);

task.add_constraints(
  {legate::align(part1, part2), legate::broadcast(part1, legate::tuple<std::uint32_t>{0})});

See also

constraints â€“ The constraints to add.
